## STYLE GUIDE NOTES

**Color Palette for AI Generation:**
- Primary: Pure black (#050505), Charcoal (#1A1A1A)
- Accent lighting: Warm amber, cool white LEDs
- Equipment: Matte black, brushed steel, chrome
- Plates: Eleiko competition colors (red, blue, yellow, green)

**Mood Keywords:**
- Industrial luxury
- Private sanctuary
- Tech-forward
- Premium minimalist
- Cinematic noir

**Avoid:**
- Bright overhead fluorescent lighting
- Crowded gym environments
- Visible faces (privacy focus)
- Cluttered backgrounds
- Generic stock photo feel

---

## RECOMMENDED GENERATION SETTINGS

**Midjourney:**
- Add `--ar 16:9` for hero/feature images
- Add `--ar 1:1` for location cards
- Use `--style raw` for more photorealistic results
- Add `--v 6` for latest model

**DALL-E 3:**
- Request "photorealistic" explicitly
- Specify "no text or watermarks"

**Stable Diffusion XL:**
- Use realistic photography LoRAs
- Negative prompt: "cartoon, illustration, text, watermark, low quality, bright lighting, crowded"
